ul : unorder list 태그 : 순서 없는(순서번호 글머리) 목록 태그

ol : ordered list 태그 : 순서 있는(순서번호 글머리) 목록 태그

li : list item 태그 : 목록 값

  • 어린 개를 일컫는 순우리말이다. 소의 새끼 → 송아지, 말 → 망아지의 어법을 따라 만들어진 용어다.
  • 강아지를 줄여 강쥐라고 부르는 경우도 존재한다.
  • 예로부터 쓰던 말은 아니고, 젊은 층에서 장난으로 영어처럼 꼬아서 발음하는 것. 예를 들어 여자친구의 막내 엄지를 팀원들이 갱얼쥐라고 불렀다.
  • 아지는 옛날에 아기를 다르게 부르는 말이다. 지방, 지역마다 조금씩 다르게 부르기도 한다.
